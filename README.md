# 왁뿌볼 공장 — 왁뿌랜드

왁뿌랜드의 첫 번째 콘텐츠, **왁뿌볼 공장** 정적 웹사이트입니다.
속재료·색상·토핑·코팅·기분을 골라 나만의 왁뿌볼을 만들고, 결과 카드로
감상한 뒤, 손끝으로 시원하게 부숴보는 온라인 놀이 시뮬레이터예요.

## 파일 구조

```
malang-lab/
├── index.html   # 메인 — 왁뿌볼 공장 소개 및 시작 버튼
├── maker.html   # 나만의 왁뿌볼 만들기 (선택 + 실시간 미리보기)
├── result.html  # 완성된 왁뿌볼 결과 카드
├── smash.html   # 왁뿌볼 부수기 게임
├── about.html   # 소개 및 이용 안내, 주의사항
├── style.css    # 전체 스타일 (파스텔톤, 모바일 반응형)
├── script.js    # 전체 로직 (선택 상태, 점수 계산, 게임, localStorage)
└── README.md
```

## 핵심 기능

- **나만의 왁뿌볼 만들기**: 속재료, 색상, 토핑, 코팅, 오늘의 기분을 골라
  실시간으로 왁뿌볼 미리보기가 바뀌어요.
- **결과 카드**: 랜덤 생성된 이름과 함께 바삭도 · 말랑도 · 스트레스
  해소력 점수, 선택 옵션 요약을 카드 형태로 보여줘요.
- **왁뿌볼 부수기 게임**: 캔버스로 그린 유광 마블 왁뿌볼을 꾹 눌러
  껍질을 와그작 깨뜨리고, 이어서 속까지 시원하게 으깨는 프레스-홀드
  게임이에요. 만든 왁뿌볼의 색상과 바삭도·말랑도 점수가 껍질/속의
  질감과 단단함에 그대로 반영돼요. Web Audio API로 만든 절차적 ASMR
  사운드, 🧊 얼리기 · 🔊 음소거 토글, 지원 기기에서는 진동 효과까지
  더했어요.
- **localStorage 저장**: 만들기 선택값, 완성된 결과, 부수기 음소거·
  얼리기 설정이 브라우저에 저장되어 새로고침·재방문 시에도 유지돼요.
- **공유 문구 복사**: 버튼 한 번으로 왁뿌볼 카드 요약 문구를
  클립보드에 복사할 수 있어요.
- **광고 영역**: index / result / smash(부수기 완료 후) / about
  페이지에 회색 박스로 표시돼요.
- **모바일 우선 반응형 디자인**: 파스텔톤의 둥근 카드형 UI로,
  인스타 스토리에 공유하고 싶어지는 결과 카드 느낌을 살렸어요.

## 로컬에서 실행하기

별도 빌드 과정이 없는 순수 HTML/CSS/JS 정적 사이트입니다.

**방법 1 — 바로 열기**

`index.html` 파일을 브라우저로 더블클릭해서 열면 됩니다.

**방법 2 — 로컬 서버로 실행 (권장)**

```bash
python3 -m http.server 8000
```

이후 브라우저에서 `http://localhost:8000` 접속.

## 배포 방법

빌드 과정이 없는 정적 사이트라 별도 빌드 없이 파일 그대로 배포하면 돼요.

- **배포는 `git push origin main`으로만 합니다.** Netlify CLI, Vercel
  CLI 등으로 로컬에서 직접 배포하지 않아요.
- Netlify / Vercel / GitHub Pages 같은 정적 호스팅에 이 저장소를 한 번
  연결해두면, 이후 `main` 브랜치로 push할 때마다 자동으로 다시
  배포돼요.
- 빌드 명령이 필요 없으므로 빌드 커맨드는 비워두고, 배포(퍼블리시)
  디렉터리만 저장소 루트(`/`)로 지정하면 됩니다.

## 기술 스택

- HTML5 / CSS3 / Vanilla JavaScript (프레임워크·빌드 도구 없음)
- 데이터 저장: 브라우저 `localStorage`
- 외부 네트워크 요청 없음 (폰트·이미지 등 전부 인라인/시스템 자원 사용)

백엔드가 필요해지면 `docs/backend-guide.md` 문서를 참고하세요.
